Branches connect nodes together to form a power network and are used to model overhead power lines, undeground cables, transformers and interconnectors etc.

This is the unique identifier for the the branch. When creating a new branch this field can be left blank and the program will generate a unquie value for this field automatically.

Branches connect two nodes that are labelled Node1 and Node2. More than one branch can be used to connect the same nodes. Power flow will be positive if the flow is from Node1 to Node2 and negative if the flow from Node2 to Node1.

Node1 => Node2 (positive powerflow)

Node2 => Node1 (negative powerflow)

This is the type of branch.

When the two nodes are of the same voltage but in different locations then the following types of Branch are valid:-

  • HVDC (High Voltage DC line)

For the same voltages at the same location these types are available:-

  • QB (Quad Booster)
  • Series Capacitor

For different voltages at the same location then this type is valid:-

  • Transformer

Controls are model artifacts that can control power flow through the network and whos values can be optimised to achieve certain goals such as maximising power flow across boundaries.

This is a unique identifier for the control. When the control is associated with a branch then the code will be the same as the branch.

This is either a Node or Zone associated with the control. For branch controls these will be the nodes used by the branch but for other types of controls they can be a Node or a zone dependent on the type

This is the type of control. Types can be divided into two broad categories: those associated with a branches and others.

Controls associated with branches control the power flow along their associated branch and can be one of these types:-

  • QB - These model Quad Boosters
  • Series Capacitor - These model the effects of series capacitors
  • HVDC - These model HVDC links

Other types of controls are:-

  • DecInc - These directly control the demand of the two nodes they are associated with
  • InterTrip - These directly control the demand of a single node and a whole zone
  • Transfer - These control demand at the two zones they are associated with

Boundaries typically divide the network in two and are useful to show general flows around the network.

A Boundary is defined by assigning it a collection of Zones. Then all nodes that belong to the zones in the boundary are assumed to be one side of the boundary with all remaining nodes being the other side of the boundary.

The unique identifier for the boundary

The collection of geographical zones defining the boundary

A Zone is a specific geographical area containing a set of nodes. A node must belong to one and only one zone.

unique identifier for the zone

Generator models are used to specify the energy mix used to satisfy the model's demand for electricity.

Each model defines a percentage contribution from each energy source used by the generators in the model.

Type of energy source. These correspond to the different types of generators defined.

The total Maximum Entry Capacity for the given energy source.

This is the total Max TEC for each generator that uses this energy source.

If set, the contribution for this energy source will be auto calculated.

At least one energy will need to be auto calculated to ensure that demand = generation.
